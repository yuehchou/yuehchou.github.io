## AWS IoT

AWS IoT 提供將 IoT 裝置連接至其他裝置和 AWS 雲端服務的雲端服務，並且可以協助使用者將 IoT 裝置整合至基於 AWS IoT 的解決方案中。因此如果使用中的裝置可以連線到 AWS IoT，則 AWS IoT 就可以將這些裝置連線到 AWS 提供的雲端服務。

1. 創建 AWS IoT 裝置連線
2. 透過 AWS IoT SDK 推送與接收訊息
